# test-docs

Sample repository for testing the gh-doc-to-confluence GitHub App.

## Contents

- `docs/getting-started.md` — Quick start guide
- `docs/architecture.md` — System architecture with PlantUML and Mermaid diagrams
- `docs/guides/setup.md` — Detailed setup instructions

## Purpose

This repo is used to verify that:
1. Markdown files are detected and published to Confluence
2. PlantUML code blocks are rendered to PNG and embedded inline
3. Mermaid code blocks are rendered to PNG and embedded inline
4. Page hierarchy maps correctly (repo > docs > guides)
5. Relative links between docs are resolved to Confluence page links
